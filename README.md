# lesson-builder

`lesson-builder` is a block-based lesson editor for creating, organizing, previewing, importing, and exporting lesson content as JSON.

Repository: [mtslma/lesson-builder](https://github.com/mtslma/lesson-builder)

## Requirements

- Node.js 20 or newer
- npm

## Installation

Clone the repository:

```bash
git clone https://github.com/mtslma/lesson-builder.git
cd lesson-builder
```

Install dependencies:

```bash
npm install
```

## Local Setup

Start the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Notes

- The app runs locally in the browser.
- Lesson data can be imported and exported as JSON.
- Local changes are saved in browser storage during development.
